# Example Patterns

## Builder Pattern
Used when an object has multiple optional parameters.

```kotlin
val user = User.Builder()
    .setName("Suman")
    .setAge(29)
    .build()
```

---


## Observer Pattern
Used when multiple objects need automatic updates from a subject.

Example:
- YouTube subscribers getting notifications
- LiveData observers in Android
- Event listeners
